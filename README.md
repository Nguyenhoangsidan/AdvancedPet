# AdvancedPet
## Getting Started

1. **Installation:** Place compiling `AdvancedPet-1.0.jar` into your Minecraft server's `plugins` folder. Ensure you also have the required dependencies (MMOCore, MythicMobs, PlaceholderAPI) installed.
2. **Initialization:** Start or Restart your server to generate the configuration files.
3. **Configuration:** Modify the standard existing pets in `pets.yml` and adjust GUI language strings in `messages.yml`. Custom skills should be configured directly via MythicMobs.
4. **Grant Pets:** Use the in-game admin commands to start granting players their very first companions!
